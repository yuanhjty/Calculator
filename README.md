# toy-calculator

## Function
### Scientific calculator
>Operand: scientific notation supported  
>Binary operation: + - * / ^ %  
>Unary  operation: - sin cos tan asin acos atan ln lg sqrt

## Demo

### CMDApp Demo

```
$ ./Calculator
toy-calculator
Type expressions to be evaluated
Type 'quit' to quit.

>> 22+5*3^(10-8)
   ________________________________________

   22 + 5 * 3 ^  ( 10 - 8 )

   = 67
   ________________________________________

>> sin(3.14/6)
   ________________________________________

   sin  ( 3.14 / 6 )

   = 0.49977
   ________________________________________

>> acos0.5
   ________________________________________

   acos 0.5

   = 1.0472
   ________________________________________

>> 3.2e2/2^3
   ________________________________________

   3.2e2 / 2 ^ 3

   = 40
   ________________________________________

>>
```
